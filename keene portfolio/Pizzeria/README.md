# Farina & Brace

Komercyjny koncept strony dla fikcyjnej pizzerii neapolitańskiej na warszawskiej Pradze. Projekt łączy editorialową typografię, ciepłą paletę i autorskie fotografie z czytelną ścieżką do zamówienia oraz rezerwacji.

## Uruchomienie

```bash
npm install
npm run dev
```

Strona jest także w pełni statyczna i może zostać otwarta przez dowolny serwer HTTP.

## Funkcje

- responsywny layout desktop/mobile,
- filtrowane menu i działający koszyk demonstracyjny,
- formularz rezerwacji z walidacją,
- mobilna nawigacja i stały przycisk zamówienia,
- animacje z obsługą `prefers-reduced-motion`,
- semantyczny HTML, skip link i dostępne opisy,
- autorskie fotografie wygenerowane specjalnie dla marki.
