# PUNKT ZERO

Kompletna strona typu one-page dla fikcyjnego studia treningu personalnego i funkcjonalnego w Warszawie. Projekt łączy sportowy editorial, nowoczesny brutalizm i czytelną ścieżkę konwersji do treningu próbnego.

## Uruchomienie

```bash
npm install
npm run dev
```

Stronę można też uruchomić dowolnym prostym serwerem statycznym — nie wymaga backendu ani frameworka.

## Zawartość

- responsywny layout desktop/mobile,
- dostępna nawigacja i formularz,
- animacje aktywowane podczas przewijania z obsługą `prefers-reduced-motion`,
- mobilne menu,
- autorskie assety fotograficzne w lekkim formacie WebP,
- kompletne treści, oferta, grafik, opinia, FAQ i dane kontaktowe.
